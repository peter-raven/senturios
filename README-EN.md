# Mission to Senturios

This is a source code archive of the danish text adventure "Mission til Senturios" (Mission to Senturios), produced by Peter Ravnholt 1989. The programming tools used were [Borland Turbo Basic 1.0](https://www.abandonwaredos.com/abandonware-game.php?abandonware=Borland+Turbo+Basic+1.0&gid=2602). This repository is kept for historic and nostalgic reasons ;-)
